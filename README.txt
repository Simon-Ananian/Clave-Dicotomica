Este proyecto permite cargar un archivo con información de especies mediante dos tipos de algoritmo: mediante la búsqueda por arboles o mediante la búsqueda por tablas hash. Así mismo se ha demostrado que las búsquedas por arboles son mas eficientes por tablas hash. 

El Usuario puede ejecutar el programa descargándolo desde el repositorio
https://github.com/Simon-Ananian/Clave-Dicotomica.git

Para ejecutarlo se realiza lo siguiente: 
1. Se coloca este comando en cmd "git clone https://github.com/Simon-Ananian/Clave-Dicotomica.git" 
2. Al descargarlo le das cd Clave/Dicotomica
3. Ejecuta el programa en NetBeans 
4. Al cargar el programa selecciona el archivo: arboles_templados.json o familias_botanicas.json
5. Una vez cargado se despliega el arbol y se habilita el botón de buscar especie
6. Al buscar especie se habilita un modal en donde se selecciona la búsqueda por arbol o la búsqueda por hash
7. El sistema realiza la búsqueda de la especie y retorna el camino hasta llegar al nodo de dicha especie y adicionalmente indica el tiempo de búsqueda
